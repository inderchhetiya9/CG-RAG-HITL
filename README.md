# CG-RAG-HITL
Concept Guided Retrieval Augmented Generation with Human In The Loop


Website to access publicly available 4C concessions:
https://autodcr.mcgm.gov.in/CitizenSearch/CitizenSearch.aspx
To find Concession Reports, Search with Project No. (available as P-XXXX in Concession Reports and Scrutiny Reports) <br>
-> Next, go to View Applications <br>
-> Go to Notesheet under Scrutiny row <br>
-> Click on the attachment link of the Executive Engineer (EE) <br>
-> Download 4C report for the project

This repository also contains
1. Examples for the scrutiny reports (This data is confidential and not available publicly in HTML format)
2. Examples of the 4C reports, which can be downloaded from the publicly available project repository
3. Examples of the output automatically generated by the CG-RAG framework without any human intervention for the same projects are also given
